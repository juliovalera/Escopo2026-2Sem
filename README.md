Escopo Sequência de Programação e Tecnologia e Inovação para o Segundo Semestre 2026

# Central de Consulta Pedagógica

Aplicação web desenvolvida para facilitar a consulta, a organização e a cópia de informações pedagógicas relacionadas aos componentes de **Tecnologia e Inovação** e **Programação** no segundo semestre de 2026.

A ferramenta permite pesquisar aulas, objetivos de aprendizagem, conteúdos, habilidades, unidades e períodos, além de copiar os dados de forma organizada para apoiar o preenchimento do **Guia de Aprendizagem**.

## Aviso importante

Esta aplicação possui caráter exclusivamente pedagógico e de apoio ao planejamento docente.

O uso da ferramenta **não desobriga os professores de consultarem a documentação oficial**, as orientações da Secretaria da Educação do Estado de São Paulo, os materiais disponibilizados nos canais institucionais e eventuais atualizações publicadas pelos órgãos responsáveis.

Em caso de divergência entre as informações apresentadas nesta aplicação e a documentação oficial, **prevalecem sempre os documentos e orientações oficiais**.

## Documentação oficial

Os documentos oficiais utilizados como referência podem ser consultados no endereço abaixo:

[Consultar a documentação oficial](https://drive.google.com/drive/folders/1OMcxZAFp9ANm7wFyDvIul0gdmtXTFoj6)

## Funcionalidades

A aplicação oferece:

* pesquisa por palavra-chave;
* filtros por componente curricular;
* filtros por ano ou série;
* filtros por bimestre, semana e unidade;
* consulta de objetivos de aprendizagem;
* consulta de conteúdos e habilidades;
* cópia individual de cada campo;
* cópia completa para o Guia de Aprendizagem;
* ajuste do tamanho das letras;
* modo de alto contraste;
* funcionamento em computadores, tablets e celulares;
* funcionamento sem banco de dados.

## Tecnologias utilizadas

O projeto foi desenvolvido com:

* HTML;
* CSS;
* JavaScript;
* arquivos JSON para armazenamento dos dados.

Por ser uma aplicação web estática, pode ser publicada gratuitamente em serviços como GitHub Pages, Netlify ou Cloudflare Pages.

## Como utilizar

1. Acesse a aplicação pelo navegador.
2. Selecione o componente curricular.
3. Escolha o ano ou a série.
4. Utilize os filtros de bimestre, semana, unidade ou aula.
5. Consulte os dados apresentados.
6. Utilize os botões de cópia para transferir as informações ao Guia de Aprendizagem.
7. Antes de finalizar qualquer documento pedagógico, confira as informações na documentação oficial.

## Atualização dos dados

Os dados da aplicação foram organizados a partir das planilhas de escopo e sequência disponibilizadas para o ano letivo de 2026.

Sempre que houver atualização da documentação oficial, recomenda-se revisar também os arquivos de dados desta aplicação.
